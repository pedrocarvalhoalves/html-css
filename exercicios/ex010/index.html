OBSERVAÇÕES IMPORTANTES

quando estou me referindo a um link interno, caso seja a próxima página do meu site por exemplo
devo adicionar rel="next" para especificar que estou direcionando-me para a próxima página.

caso eu esteja na segunda página e tenha um link para retornar a página 1, uso rel="prev" para
especificar que estou voltando para a página anterior.

na terceira página do meu site, disponibilizei um link para retornar a essa página. porém...
esta página está em outra pasta. quando quero retornar à outra pasta eu digito no href dessa forma: ../
dois pontos barra e então me será apresentado opções de link para adicionar.
