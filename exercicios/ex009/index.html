aprendi a fazer listas ordenadas(onde a ordem importa, obviamente) e listas desordenadas, onde a ordem tanto faz.

listas ordenadas

  1. html
  2. css
  3. javascript
  4. aprofundar em backend

essa lista ficou ordenada numericamente, mas também posso optar por organizar em ordem alfabética, seja maiúscula ou minúscula, e também em algarismo romano, seja maiúsculo ou minúsculo.
ficaria assim <ol type=1, A, a, I, i

  1. vai
  2. tomar
  3. no
  4. cu

também posso escolher de onde começar, por exemplo:
<ol type=1 start=3
e aconteceria o seguinte:
  1. html
  2. css
  3. javascript
  4. aprofundar em backend

lista desordenada

  • pao
  • queijo
  • carne
  • ovo
  • maionese
  • ketchup

para ordenadas, <ol>, para desordenadas, <ul>

listas mistas

posso misturar as listas, colocar ordenada dentro de ordenada, desordenada dentro de ordenada e muito mais.

  1. este é um exemplo
    1. este é outro exemplo
    2. esse exemplo aqui
    3. é com lista ordenada
  2. +1 exemplo
    1. esse exemplo aqui
    2. é trocando ordem númerica
    3. por ordem alfabética
  3. + outro exemplo
    • essa aqui
    • simplesmente
    • não
    • tem
    • ordem

listas de definição

listas de definição são básicamente termo e descrição.
para abrir lista de definição, uso <dl>.
termo = <dt>
descrição = <dd>

exemplo de lista de definição

HTML
linguagem de marcação para a criação de conteúdo de um site
CSS
linguagem de marcação para a criação de design de um site
Javascript
linguagem de programação para a criação de interatividade
