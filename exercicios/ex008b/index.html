outras formatações

código fonte / identação

se eu quero citar um código em python por exemplo...
print('Olá, Mundo!) num = int(input('Digite um número: '))
a fonte do site não é a mais adequada para ler códigos, a tag <code> para usar uma fonte mono-espaçada.

print('Olá, Mundo!) num = int(input('Digite um número: ')) num2 = int(input('Digite outro número: '))
agora a fonte está mono-espaçada, porém identar o código deixaria ele mais compreensível visualmente.
pra isso usamos a tag <pre> antes da tag <code>.


        print('Olá, Mundo!)
        num = int(input('Digite um número: ')) 
        num2 = int(input('Digite outro número: '))

citação completa

para fazer uma citação completa usa-se <blockquote>

Segundo o filme American Pie - Caindo em Tentação.

Ele é um aluno nota 10, exceto pelo seu hábito de ficar bêbado deliberadamente, apagar, causar uma completa e total zona, e nunca lembrar de uma coisa se quer.
